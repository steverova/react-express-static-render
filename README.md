# Vite Static Render

A project for static rendering with Vite, React, and Express.

## Libraries Used

### Dependencies
- **@hookform/resolvers**: Resolvers for React Hook Form to integrate with validation libraries like Zod.
- **@tanstack/react-query**: Powerful data synchronization for React applications, handling server state.
- **@tanstack/react-table**: Headless UI for building powerful tables and datagrids.
- **@vite-static-render/shared**: Local shared utilities and types for the project.
- **argon2**: Secure password hashing library.
- **async-handler**: Utility for handling async errors in Express routes.
- **axios**: Promise-based HTTP client for the browser and Node.js.
- **bcrypt**: Library for hashing passwords.
- **compression**: Middleware for gzip compression in Express.
- **cors**: Middleware for enabling CORS in Express.
- **dotenv**: Loads environment variables from a .env file.
- **express**: Fast, unopinionated web framework for Node.js.
- **express-rate-limit**: Middleware for rate limiting in Express.
- **helmet**: Security middleware for Express applications.
- **jose**: JavaScript library for JSON Web Tokens (JWT).
- **lucide-react**: Beautiful & consistent icon toolkit made by the community.
- **morgan**: HTTP request logger middleware for Node.js.
- **nanoid**: Tiny, secure, URL-friendly unique string ID generator.
- **rate-limit**: Rate limiting utility.
- **react**: A JavaScript library for building user interfaces.
- **react-dom**: React package for working with the DOM.
- **react-dropzone**: Simple React hook to create a drag-and-drop zone.
- **react-hook-form**: Performant, flexible forms with easy validation.
- **react-router**: Declarative routing for React.
- **react-virtualized**: React components for efficiently rendering large lists and tabular data.
- **styled-components**: Visual primitives for the component age, using CSS-in-JS.
- **winston**: Versatile logging library for Node.js.
- **zod**: TypeScript-first schema declaration and validation library.
- **zustand**: Small, fast and scalable state management for React.

- **tsx**: TypeScript executor for Node.js.
- **typescript**: Typed superset of JavaScript.
